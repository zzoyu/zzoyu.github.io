# zzoyu.github.io

Vue 3으로 작성된 소개 페이지입니다. 프로젝트 관련 업데이트는 2024년 기준으로 작성되어 있습니다.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
